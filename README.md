# 💸 Smart Expense Tracker

A clean and interactive **Expense Tracker Web App** built using HTML, CSS, and JavaScript.

This project focuses not just on functionality, but also on **usability, UI design, and responsiveness**.

---

## 🚀 Features

- 💰 Set and manage monthly budget  
- 📊 Real-time expense tracking  
- 🧾 Add & delete expenses easily  
- 🔍 Filter expenses by category  
- 📈 Interactive doughnut chart (Chart.js)  
- 🌙 Dark / Light mode toggle  
- 🎨 Custom UI (including Gojo theme toggle 😄)  
- 📱 Fully responsive (works on mobile + desktop)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Chart.js  
- LocalStorage
